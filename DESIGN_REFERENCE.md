## ‚ú® Key Design Principles

### Visual Style
- **Light & Clean**: White/light gray backgrounds
- **Bold Typography**: Extra-large, bold headings (36-48px)
- **Pastel Cards**: Functional cards with soft pastel backgrounds
- **Flat Design**: No shadows, gradients, or neumorphism
- **Minimalist Icons**: Simple black line icons
- **Generous Whitespace**: Breathing room between all elements

### Color Palette

#### Backgrounds
- Primary: `#FFFFFF` (White)
- Secondary: `#F5F5F7` (Light Gray)
- Tertiary: `#E8E8ED` (Lighter Gray)

#### Pastel Card Colors
- Blue: `#D4E7F7` (Light Blue)
- Beige: `#F5EFE7` (Cream)
- Green: `#E3F5E8` (Mint Green)
- Yellow: `#FFF9E3` (Light Yellow)
- Pink: `#FFE8F0` (Light Pink)
- Purple: `#EDE7F6` (Lavender)

#### Functional Colors
- Success: `#34C759` (Green)
- Warning: `#FF9500` (Orange)
- Error: `#FF3B30` (Red)
- Info: `#007AFF` (Blue)

#### Text & Accents
- Text Primary: `#000000` (Black)
- Text Secondary: `#6B6B6B` (Gray)
- Text Tertiary: `#999999` (Light Gray)
- Accent Black: `#000000`

### Typography
- **Font**: SF Pro Display (iOS) / Roboto (Android)
- **Display**: Bold, 36-48px (Very prominent hero text)
- **Headings**: Bold, 28-32px
- **Subheadings**: SemiBold, 20-24px
- **Body**: Regular, 16-17px
- **Captions**: Regular, 13-15px

### Components

#### Cards
- Pastel backgrounds (blue, beige, green, yellow, pink, purple)
- Rounded corners: 20-24px
- No shadows or elevation
- Clean, flat appearance

#### Buttons
- **Primary**: Black circular buttons with white icons
- **Secondary**: White buttons with black borders
- **Text**: Simple black text links
- Rounded corners: 16-20px

#### Icons
- Simple, minimalist black line icons
- Consistent stroke width
- Clear and recognizable

#### Bottom Navigation
- Black circular FAB (Floating Action Button)
- Simple icon-based navigation
- Black icons when active, gray when inactive

## üìê Layout Principles

### Spacing
- Generous padding: 20-24px
- Card spacing: 16-20px
- Element spacing: 12-16px

### Grid
- 2-column grid for cards (where applicable)
- Horizontal scrolling for galleries
- Vertical stacking for lists

### Hierarchy
- Extra-large bold headings grab attention
- Pastel cards group related functionality
- Clean separation between sections
- Minimal visual noise

## üéØ Design Goals

1. **Clarity**: Information is easy to find and understand
2. **Simplicity**: No unnecessary elements or decorations
3. **Consistency**: Uniform design language throughout
4. **Accessibility**: High contrast text, readable fonts
5. **Modern**: Contemporary iOS-inspired aesthetic
6. **Delightful**: Smooth animations and pleasant interactions
